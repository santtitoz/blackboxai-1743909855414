# BarberStyle - Barbearia Profissional

## Project Overview
BarberStyle is a professional barber shop website that showcases the services, portfolio, and contact information of the business. Designed with modern web standards, this project uses a combination of HTML, CSS, and JavaScript along with frameworks like Tailwind CSS and Font Awesome for styling and icons. The site aims to reflect the quality and professionalism of the barbering services offered.

## Installation
To set up the BarberStyle project locally, you need to follow the steps below:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/barberstyle.git
   cd barberstyle
   ```

2. **Open the HTML file**:
   Open `index.html` in your web browser to view the project.
   Alternatively, you can host the project with a local server for better compatibility, for example, using Live Server extension in VS Code or any local development server of your choice.

## Usage
- Visit the homepage to view the hero section, which features the shop's tagline and a call to action to book an appointment.
- Navigate through different sections including About, Services, Portfolio, and Contact.
- Use the appointment section to schedule a visit to the barbershop.

## Features
- Responsive design that adjusts for different screen sizes.
- Smooth navigation through the use of Tailwind CSS for styling.
- Showcase of services offered with detailed descriptions.
- Portfolio section displaying high-quality images of the barber's work.
- Contact information and working hours provided for customer convenience.

## Dependencies
The project includes the following external libraries and frameworks:
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Font Awesome**: Icons for better user interface representation.
- **Google Fonts**: Custom font inclusion for better typography.

These dependencies are linked in the `index.html` file, needing an active internet connection to load them.

## Project Structure
The project is organized into the following structure:

```

├── css
│   └── styles.css          # Custom CSS styles
└── js
    └── main.js            # JavaScript for interactive features
```

## Conclusion
This project serves as a foundation for a modern barber shop website. Feel free to modify and enhance it according to your preferences and needs. For contributions, please open an issue or create a pull request.
